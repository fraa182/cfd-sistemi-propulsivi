# CFD Sistemi Propulsivi (La Rocca)
Repository del progetto di gruppo 

# Come funziona il tutto
Ogni volta che facciamo esercitazione o che si modifica il codice, chi modifica il codice deve caricarlo nella repository facendo "add file" e poi "upload files". In questo modo evitiamo che si incasini tutto e possiamo gestire gran parte delle cose da qua.
